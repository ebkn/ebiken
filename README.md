# portfolio

[blog.ebiken.dev](https://blog.ebiken.dev)

## Development

### Setup
```sh
$ npm i
```

### start server
```sh
# development
$ npm run start

# production
$ npm run serve
```

### create blog template
```sh
$ FILENAME=<blog-title> npm run create-blog
```
then, edit content/blog/$blog-title/index.md.

### run linter
```sh
$ npm run lint
$ npm run lint:fix
```

### build
```sh
$ npm run build
```
